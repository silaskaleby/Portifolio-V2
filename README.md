## 📖 Sobre o projeto

Este portfólio foi desenvolvido com o objetivo de reunir, em um único lugar, um pouco sobre minha trajetória como desenvolvedor, as tecnologias que utilizo e os projetos que venho construindo.

A aplicação também funciona como uma forma de apresentar meu trabalho para recrutadores, empresas, desenvolvedores e pessoas interessadas em conhecer minhas habilidades.

O projeto foi construído com foco em:

- Design moderno e agradável;
- Navegação simples e intuitiva;
- Responsividade para diferentes dispositivos;
- Apresentação clara das minhas habilidades;
- Divulgação dos meus principais projetos;
- Facilidade de contato.
